Spring Boot Actuator

Spring Boot Actuator is a subproject of Spring Boot that provides a set of endpoints to monitor and manage your Spring Boot applications. It allows you to inspect various aspects of your application's runtime behavior, such as:
  • Health: Checks the overall health of your application.
  • Metrics: Exposes performance metrics like HTTP requests, memory usage, and garbage collection.
  • Info: Displays application-specific information like version and build details.
These endpoints can be accessed through HTTP requests or other mechanisms like JMX. Actuator is a valuable tool for developers and operations teams to gain insights into their applications and troubleshoot issues.
